
# Neko Atom Syntax #

A subtle dark colored syntax theme for Atom based on the Seti color scheme.

![](https://raw.githubusercontent.com/catlinman/neko-atom-syntax/master/preview.png)

## Neko Atom UI ##

The corresponding UI theme can be found at
[Neko Atom UI](https://atom.io/packages/neko-atom-ui).
